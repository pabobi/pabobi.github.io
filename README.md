## 🚀 如何使用

### 方式一：直接在浏览器打开
双击 `index.html` 即可在浏览器中预览。

### 方式二：本地服务器（推荐）
```bash
# Python 3
python -m http.server 8000

# 然后访问 http://localhost:8000
```

### 方式三：部署到云端
- **GitHub Pages**：上传整个文件夹到 GitHub 仓库 → Settings → Pages → 启用
- **Vercel**：直接拖拽文件夹到 vercel.com
- **Netlify**：同上，拖拽即可

## 🔧 如何添加照片和证书

网站设计为可交互的，直接点击占位区域即可上传图片：

1. **个人照片**：点击 Hero 区照片占位框 → 选择本地照片（建议 1:1 正方形或 3:4 竖版）
2. **证书图片**：点击证书展示区的灰色占位框 → 上传证书照片
3. **项目图片**：点击项目展示区的灰色占位框 → 上传项目现场照片

> 💡 提示：所有图片上传均为本地预览，不会上传到任何服务器。

## ✏️ 如何修改内容

直接在 `index.html` 中编辑文字内容，关键位置说明：

- **个人信息**：搜索 `申万山` 替换为实际姓名
- **联系方式**：搜索 `18209781355` / `774071052@qq.com` 替换
- **工作经历**：在 `<section id="experience">` 中编辑各时间段内容
- **项目经验**：在 `<section id="portfolio">` 中增删项目
- **资格证书**：在 `<section id="certificates">` 中补充证书信息
